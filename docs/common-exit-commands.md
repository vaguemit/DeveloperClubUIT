# Common Exit Commands

## Terminal/Command Prompt
```bash
exit
```

## Vim Editor
```
:q       # quit
:q!      # force quit without saving
:wq      # save and quit
```

## Git Interactive Mode
```bash
:q       # quit
q        # quit
Ctrl + C # abort/exit
```

## Node.js REPL
```
.exit
# OR
Ctrl + C (twice)
# OR
Ctrl + D
```

## VS Code
- Windows/Linux: `Ctrl + Q`
- Mac: `Cmd + Q`
